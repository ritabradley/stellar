# Stellar

A single page web app that lets you take an image URL and upload it to the interface where our face recognizer will detect any faces in the image.

## Getting Started

Join us and register with an email and password.

## How To Use

Then find an image with a face or several faces and copy the image link. Paste the link into the input box and click 'Detect'. Voila! Faces detected!
